# Mehrdad Moradi — Portfolio

Personal portfolio of **Mehrdad Moradi**, Senior Product Manager with a developer's
foundation — 11+ years across gaming, Web3, and immersive XR.

A single, self-contained `index.html` — no build step, no dependencies.

- **Bilingual** — English / فارسی, with a language toggle (defaults to English, RTL for Persian)
- **Light & dark themes**
- **Fully responsive**, accessible, and animation-aware (respects `prefers-reduced-motion`)

## Deploy (GitHub Pages)

1. Push this folder to a GitHub repo.
2. In **Settings → Pages**, set the source to the `main` branch, root (`/`).
3. The site is served from `index.html`. `.nojekyll` tells Pages to serve files as-is.

`Mehrdad-Moradi-Resume.pdf` is linked from the page (the "Download Résumé" button) — keep it alongside `index.html`.

## Local preview

Any static server works, e.g.:

```bash
python -m http.server 5179
# then open http://localhost:5179
```
